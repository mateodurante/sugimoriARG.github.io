Cómo elegir un pokemon inicial:

Cómo elegir un pokemon inicial: Cada mazo del tablero (Bosque, Montaña, Isla, Llanura y Pantano) contiene un posible PKM inicial. Bosque: BULBASAUR (más una carta de elemento Planta) Montaña: CHARMANDER (más una carta de elemento Fuego) Isla: SQUIRTLE (más una carta de elemento Agua) Llanura: MACHOP (más una carta de elemento Lucha) ( Pantano: ABRA (más una carta de elemento Psíquico) O Cada PKM inicial trae equipado un elemento de su respectivo tipo (planta, etc) para poder realizar al menos un ataque. Tu PKM inicial te protegerá de los PKM salvajes y con el tiempo podrá evolucionar para enfrentar a los Líderes de Gimnasio y ganar Medallas.
